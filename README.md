# Building a Second Brain through Note-Taking — the Obsidian + PARA way

This repository provides a ready-to-use Obsidian vault structure based on the **PARA method** (Projects, Areas, Resources, Archive).

---

## ⚡ Quick Start (5 minutes)

### For Non-Technical Users:

1. **Download the vault**: 
   - Click the green "Code" button at the top of this page
   - Select "Download ZIP"

2. **Extract the ZIP file**:
   - Right-click the downloaded ZIP file
   - Select "Extract All" (Windows) or double-click (Mac)

3. **Open in Obsidian**:
   - Open the Obsidian app
   - Click "Open folder as vault"
   - Navigate to and select the extracted folder
   - Click "Open"

4. **Done!** Your vault is ready to use. Skip to "Setup Daily Notes" below.

---

## 📁 What's Included

The vault comes with this folder structure:

- **0 Inbox** — Quick capture area for new notes and ideas
- **1 Projects** — Active projects you're working on
- **2 Areas** — Ongoing areas of responsibility (includes Journaling folder)
- **3 Resources** — Reference materials (DevOps, AI, Learning, etc.)
- **4 Archive** — Completed projects and old notes
- **Templates** — Pre-made templates for consistent note-taking
  - `Daily Template.md` — Template for daily journaling

---

## ⚙️ Setup Daily Notes (Optional but Recommended)

To enable automatic daily note creation with the provided template:

1. **Enable Daily Notes plugin:**
   - Open Obsidian Settings
   - Go to "Core plugins"
   - Find "Daily notes" and toggle it ON

2. **Configure Daily Notes:**
   - Go to Settings → "Daily notes"
   - Set the following:
     - **New file location:** `2 Areas/Journaling`
     - **Date format:** `YYYY-MM-DD`
     - **Template file location:** `Templates/Daily Template`

3. **Create your first daily note:**
   - Click the calendar icon in the sidebar (or use the command palette)
   - A new daily note will be created automatically with the template

## 💡 What is PARA?

PARA is a simple organizational system created by Tiago Forte:

- **Projects** — Specific outcomes you want to achieve (time-bound)
- **Areas** — Ongoing responsibilities you want to maintain
- **Resources** — Reference materials and resources
- **Archive** — Completed items and past projects

This system keeps your knowledge organized and actionable.
