# Building a Second Brain through Note-Taking — the Obsidian + PARA way

This repository provides an Obsidian vault structure based on the PARA method:

- 0 Inbox
- 1 Projects
- 2 Areas (includes Journaling)
- 3 Resources (DevOps, AI, Learning)
- 4 Archive
- Templates (top-level)

## Daily Notes setup (recommended)
In Obsidian:
1) Settings → Core plugins → enable **Daily Notes**
2) Settings → Daily Notes:
   - New file location: `2 Areas/Journaling`
   - Date format: `YYYY-MM-DD`
   - Template file location: `Templates/Daily Template`

## Notes
This repo is intended as a public starter structure + templates (no private notes).
